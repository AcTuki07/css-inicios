INICIOS DE CSS

Es una herramienta que se usa para creaar paginas de internet con un diseño atractivo y agradable para el usuario.

Funciona aplicando estilos a los elementos de la pagina, como colores, tamaños, fuentes, etc. Los estilos se definen en un archivo aparte llamado hoja de estilos.

Instrucciones basicass

  1. Seleccionar el elemento al que se le quiere aplicar el estilo, usando al selector de etiqueta (div, p, h1,), de clase (.mi-clase), de ID (#mi-id), de atributo ([type="text"]), selector universal (*), pseudo-clases (a:hover), pseudo-elementos (p:first-line) y combinadores (descendiente)
  2. Definir el estilo que se le quiere aplicar
  3. Asociar el estilo al elemento

Modelo de cajajas

Las cajas son los elementos fundamentales de una pagina web. Cada caja tiene un contenido, un borde y un margen.

Hay diferentes modelos de cajas:

  • Modelo de caja margin: Area al rededor del border
  • Modelo de caja de border: Es como una frontera que rodea el padding y el contenido
  • Modelo de caja de padding: Rodea el contenido
  • Modelo de caja de contenido: Donde se muestra el texto o las imagenesz

Asociar una hoja de estilos a un documento

Hay 3 maneraas de las cuales se le pude hacer uso a CSS

  1. Estilos externos: Se definen en un archivo aparte con extension .css (ejemplo: style.css) y se asocia al documento con la etiqueta link, es el mas adecuado, recuerde ligar el arcivho .css al html usando <"link rel="stylesheet" href="./style.css">

  2. Estilos internos: Se definen en la etiqueta style dentro del head del documento, todo lo que se escriba dentro de esta etiquieta en el head, sera CSS

  3. Estilos en linea: Se agreaga al elemento el atributo style. Por ejemplo, en el cuadrado de arriba < "div style=" background-color: aqua; height: 200px; width: 200px; padding: 12px; margin: 40px; border: 10px solid red;" div>

Selectores

Los selectores son las reglas que se usan para aplicar estilos a los elementos de la pagina.

Selector de ID

Selecciona un elemento de un ID (Ejemplo: "< h2 id="subtitulo rojo">)"

Ejemplo de como se usa

<"style>
#subtitulo-rojo {
color: red;
}
style>


Para poder usar el selector de ID, se debe de poner un # antes del nombre del ID

Selector Clases

Selecciona todos los elementos de una clase (Ejemplo: "< h2 class="subtitulo-rojo">)"

Ejemplo de como se usan

<"style>
.subtitulo-rojo {
color: red;
}
style>


Para poder usar el selector de clase, se debe de poner un . antes del nombre de la clase

Medidas

Hay diferentes tipos de medidas que se pueden usar en CSS

Unidades de longitud absolutas: Se definen en relacion a una medida fija

  1. cm: Centimetros
  2. mm: Milimetros
  3. in: Pulgadas
  4. px: Pixels
  5. pt: Puntos
  6. pc: Picas

Unidades de longitud relativas: Se definen en relacion a un elemento

  1. em: Tamaño de la fuente del elemento
  2. ex: Altura de la x de la fuente del elemento
  3. rem: Tamaño de la fuente del elemento raiz
  4. vw: 1% del ancho de la ventana
  5. vh: 1% de la altura de la ventana
  6. %: Porcentaje del tamaño del elemento padre

Elemeto de bloque < div>

Es un contenedor de bloques, se usa para dividir la pagina en secciones

Se puede usar para:

  • Dividir la pagina en secciones
  • Crear columnas
  • Crear un layout

Posicionamiento

"position" es como una regla que permite posicionar los div. Hay diferentes maneras de posicionar los elementos en una pagina web

  1. Posicionamiento static: Los elementos se colocan en el orden en el que aparecen en el codigo HTML, estas propiedades no se ven afectados por las propiedades left, right, top y buttom
  2. Posicionamiento relative: Los elementos se colocan en relacion a su posicion original
  3. Posicionamiento absolut: Los elementos se colocan en relacion a su contenedor mas cercano, por defecto tomara el body como valor relativo, pero si el contenedor padre es relativo, tomara el contenedor padre como referencia.
  4. Posicionamiento fixed: Los elementos se colocan en relacion a la ventana del navegador, es decir, aunque usted haga scroll, el elemento seguira en la pantalla; puede ser util para navigation bar (Nav Bar)
